# MekanismDelight
![](https://github.com/Mochi-753/MekanismDelight/blob/main/docs/Logo.png)

**MekanismDelight** is a small mod that adds cooperative elements between Farmer's Delight and Mekanism.  
[GitHub](https://github.com/mochi-753/MekanismDelight)

## Features

**MekanismDelight** works simply by installing it.  
Integration features are provided for **Mekanism**, **Mekanism: Generators**, and **Mekanism: Tools**.  
If any Mekanism module is not installed, the corresponding integration features will be automatically disabled.
