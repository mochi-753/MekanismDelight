# MekanismDelight
<img src="https://github.com/Mochi-753/MekanismDelight/blob/main/docs/Logo.png?raw=true" alt="Logo">

**MekanismDelight** is a small mod that adds integration features between **Farmer’s Delight** and **Mekanism**.  
[GitHub Repository](https://github.com/mochi-753/MekanismDelight)

## Features

**MekanismDelight works out of the box—just install it.**  
Integration features are provided for **Mekanism**, **Mekanism: Generators**, and **Mekanism: Tools**.  
If any Mekanism module is not installed, the corresponding integration features will be automatically disabled.
